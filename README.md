간단한 그림 그리기 도구입니다.

[Tab]버튼을 통해 [검정, 하양, 빨강, 초록, 파랑, 시안, 마젠타, 노랑] 8가지 색깔을 사용할수 있습니다.

[+]버튼과 [-]버튼을 통해 붓의 두께를 조정할 수 있습니다.

[Esc] 버튼을 통해 창을 종료할 수 있습니다.
